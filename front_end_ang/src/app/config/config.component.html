Geração do Token de segurança para acesso à API

Este token de segurança é uma sequencia de letras e/ou números que precisa ser adicionada na URL para acessar as APIs do sistema.

Nome de usuário
Senha
Validade do token (em minutos)


Token de segurança gerado

Este token de segurança deve ser usado an URL quando for acessar uma API do sistema a partir de outro programa/sistema.
Por exemplo:
http://servidor:porta/produto_lista/fffffffffffffffffffff
onde servidor deve ser o nome ou ip do servidor, porta deve ser o número da porta usada pelo sistema e fffffffffffffffffffff é o token de segurança gerado pelo sistema.
